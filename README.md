# Esther Tsotso Portfolio

A personal portfolio for Esther Bawo Tsotso, a product-minded frontend engineer focused on solving real user and product problems with thoughtful, accessible interfaces.

The site highlights selected projects, technical tools, experience, education, awards, and contact links in a responsive React portfolio with dark/light mode, sticky project cards, sound controls, and smooth interaction details.

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
